# Curso de Programación Orientada a Objetos en Python

Este repositorio contiene ejemplos prácticos y ejercicios del curso de Programación Orientada a Objetos (POO) en Python.

## Temas Cubiertos

### 1. `topico1-clases`
Conceptos fundamentales de clases en Python:
- Atributos de clase vs atributos de instancia
- `@dataclass` para estructuras de datos
- DTOs (Data Transfer Objects)

### 2. `topico2-herencia`
Herencia simple y múltiple en Python:
- Herencia de clases
- Herencia múltiple
- MRO (Method Resolution Order)

### 3. `topico3-polimorfismo`
Diferentes formas de implementar polimorfismo:
- Override de métodos
- Métodos abstractos con `@abstractmethod`
- Interfaces y abstracción


### 4.`topico4-encapsulamiento`
Control de acceso a los datos:
- Atributos protegidos (`_`) y privados (`__`)
- Getters y setters
- Decorador `@property`
- Decoradores personalizados

### 5.`topico5-abstraccion`
Ocultar complejidad:
- Ocultar detalles de implementación
- Clases abstractas con `ABC`
- Métodos abstractos obligatorios


### 6. `topico6-funciones-especiales`
Métodos mágicos en Python:
- `__str__` para representación legible
- `__repr__` para representación técnica
- Sobrecarga de operadores (`__add__`, etc.)

### 7. `topico7-SOLID`
Principios de diseño de software:
- **SRP**: Single Responsibility Principle
- **OCP**: Open/Closed Principle
- **LSP**: Liskov Substitution Principle
- **ISP**: Interface Segregation Principle
- **DIP**: Dependency Inversion Principle


## Requisitos

- Python 3.8+
- No requiere dependencias externas


## Contenido de Cada Archivo

Cada archivo contiene teoría explicada en comentarios y ejemplos de código funcionales que demuestran los conceptos de cada tema.
